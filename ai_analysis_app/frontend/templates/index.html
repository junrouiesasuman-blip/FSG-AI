About the System

What FSG AI Does

FSG AI processes user-submitted information and generates intelligent predictions through a trained Random Forest model.

Risk Assessment

Evaluates input data and determines whether the condition is Normal, High Risk, or Anomaly.

Anomaly Detection

Identifies unusual patterns or abnormal values that may require further attention and monitoring.

Prediction History

Stores previous analysis results so users can review and track recent predictions anytime.

System Workflow

How the System Works

The system follows a structured process from secure user access to AI-powered prediction and result storage.

User Authentication

Users securely create an account or log in before using the system.

Data Submission

The user enters the required analysis values into the prediction form.

AI Processing

The trained Random Forest model processes the submitted data.

Prediction Result

The system classifies the result as Normal, High Risk, or Anomaly.

Result Review

Users can immediately view the prediction result and confidence level.

History Tracking

All previous analyses are stored and accessible through the history page.
